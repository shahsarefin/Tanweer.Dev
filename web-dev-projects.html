AdoptADog
Description

AdoptADog is a web application designed to streamline the dog adoption process. It serves as a platform where users can explore various dog breeds, learn about their unique characteristics, and find dogs available for adoption.

Technologies Used
  • Ruby on Rails
  • Bootstrap
  • Docker
  • Fly.io (for deployment)
Key Features
  • Browse detailed list of dog breeds
  • In-depth breed information pages
  • Simple and Hierarchical Search functionality
  • User profiles with contact details
  • Adoption process information
Challenges and Solutions

One of the main challenges was integrating multiple data sources and ensuring data consistency across the application. This was solved by implementing robust data import and validation processes.

Lessons Learned

This project deepened my understanding of Ruby on Rails, particularly in areas of data modeling and associations. It also provided valuable experience in working with external APIs and handling data imports.

Movie.Info
Description

Movie.Info is a web application that provides users with information about popular, top-rated, and upcoming movies. It features a search functionality that allows users to find movies quickly and view detailed information.

Technologies Used
  • React
  • TMDB API
  • Styled-components
Key Features
  • Displays popular, top-rated, and upcoming movies
  • Responsive design for different devices
  • Search functionality to find specific movies
Challenges

Managing multiple API calls efficiently was a key challenge, which was addressed by creating custom hooks for data fetching.

Lessons Learned

This project enhanced my skills in state management and working with APIs, particularly in managing data efficiently in React.

Personal Portfolio Website
Description

A comprehensive personal portfolio website designed and developed from scratch. This project showcases my skills in web development, design, and project management.

Technologies Used
  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap 5
  • Lightbox
  • Formspree (for contact form functionality)
Key Features
  • Responsive design that works seamlessly on desktop and mobile devices
  • Interactive project cards with modal windows for detailed information
  • Vertical timeline showcasing professional and educational history
  • Functional contact form integrated with Formspree
  • Custom animations for enhanced user experience
Challenges and Solutions

One of the main challenges was implementing a functional contact form without a backend. This was solved by integrating Formspree, a third-party service that handles form submissions.

Lessons Learned

This project reinforced the importance of planning in web development, enhanced my skills in responsive design, and gave me practical experience in integrating third-party services.

Interactive API with Flask and Dash
Description

An interactive application featuring advanced functionalities and a user-friendly interface. This project combines Flask for backend operations and Dash for frontend data visualization.

Technologies Used
  • Flask
  • Dash
  • Pillow (for image processing)
  • Faker (for dataset generation)
  • Plotly (for data visualization)
Key Features
  • Image conversion functionality using Pillow
  • Dataset generation and download feature with Faker
  • Interactive data visualization with Plotly in Dash
  • User-friendly interface for easy navigation and interaction
Challenges and Solutions

Integrating Flask with Dash for a seamless experience posed a challenge. The solution involved careful planning of routes and data handling between Flask and Dash.

Lessons Learned

This project improved my skills in integrating backend and frontend technologies. It also enhanced my understanding of data visualization and interactive web applications.
